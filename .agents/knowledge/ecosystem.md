# Open Source Ecosystem

Ubot is a community-driven platform for decentralized, private, and agentic intelligence.

## 1. Open Source Philosophy

- **Accessibility**: Built on the MIT license to encourage experimentation and contribution.
- **Modularity**: The registry-based architecture allows developers to build and share their own **Tool Modules** and **Agent Personas**.
- **Transparency**: Every line of code is auditable, ensuring that the "Soul" of your AI remains under your control.

## 2. Tool-Sharing and MCP

The **Model Context Protocol (MCP)** is our bridge to the wider AI ecosystem.

- **Interoperability**: Any MCP server built for other agents (like Claude Desktop) works natively with Ubot.
- **Connectors**: Developers can add specialized MCP servers to extend Ubot's capabilities beyond its core modules.

## 3. Contributing

- **GitHub**: Our primary hub for development, issues, and feature requests.
- **Registry Contributions**: Owners are encouraged to share their Agent Personas (Markdown) and Tool Modules to broaden the platform's utility.
